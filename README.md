# Pac-Man

A Pac-Man game built with [Go](https://go.dev/) and [Ebiten](https://ebitengine.org/), a 2D game engine for Go.

## Prerequisites

- [Go](https://go.dev/dl/) 1.21 or later
- System dependencies for Ebiten (see [Ebiten install guide](https://ebitengine.org/en/documents/install.html))

## Getting Started

```bash
# Clone the repository
git clone https://github.com/hbraswelrh/pacman.git
cd pacman

# Run the game
go run .
```

## Controls

- **Arrow keys** — Move Pac-Man

## Building

```bash
go build -o pacman .
./pacman
```
